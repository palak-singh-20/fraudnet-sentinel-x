# 🛡️ FraudNet Sentinel X

AI-powered Cyber Resilience Platform for Critical National Infrastructure.

## 🚀 Overview

FraudNet Sentinel X is an AI-powered cybersecurity platform designed to detect, analyze, and respond to cyber threats targeting critical infrastructure. It combines intelligent fraud detection, scam analysis, incident visualization, and victim assistance into one unified platform.

## ✨ Features

- 🔍 AI Scam Detection
- 📊 Real-time Security Dashboard
- 🧠 Threat Intelligence
- 🚨 Incident Monitoring
- 🛟 Victim Rescue & Guidance
- 📈 Analytics & Reporting
